# Guitar-to-Doom Controller Pro

Превратите вашу электрогитару или бас-гитару в игровой контроллер для GZDoom! Этот Python-скрипт анализирует аудиосигнал в реальном времени и преобразует высоту тона и амплитуду в команды клавиатуры.

## Сетап GZDoom

Этот проект работает с **GZDoom v4.14.2** (портативная установка) — современным движком Doom с продвинутой графикой (OpenGL/Vulkan), поддержкой модов и высокими разрешениями.

### Установленные IWAD (игровые данные)

| Папка | Содержимое |
|-------|-----------|
| `iwads/` | doom2.wad и другие основные IWAD файлы |
| `official/` | Официальные аддоны от id Software и Nightdive Studios |

**Официальные аддоны в `official/`:**
- **id1.wad** — Unity-версия Doom 2 с новым эпизодом
- **iddm1.wad** — Deathmatch maps от id Software
- **id1-mus.wad, id1-res.wad, id1-tex.wad** — ресурсы для id1
- **nerve.wad** — No Rest for the Living (эпизод для Doom 2)
- **sigil.wad** — SIGIL (эпизод от Ромеро)
- **masterlevels.wad** — Master Levels for Doom II

### Установленные модификации

Папка `mods/` содержит улучшения для игрового опыта:

#### Аудио (`mods/audio/`)
| Мод | Описание |
|-----|----------|
| **IDKFAv2.wad** (90MB) | Impressive MIDI/Soundfont — качественный звуковой шрифт с эффектами реверберации и хора. Автоматически загружается для всех Doom-игр. |

#### Графика (`mods/graphics/`)
| Мод | Описание |
|-----|----------|
| **NeuralUpscale2x_v1.0.pk3** (132MB) | Нейросетевой апскейл текстур в 2x с сохранением стиля оригинала. Текстуры выглядят чётче, но остаются пиксельными. |

### Конфигурация запуска

Все BAT-файлы автоматически:
1. Загружают IWAD из `iwads/`
2. Подключают моды из `mods/` и `skins/`
3. Используют общий конфиг `gzdoom.ini`
4. Сохраняют в `save/`

```batch
:: Структура запуска (Play DOOM II.bat)
gzdoom.exe ^
  -config gzdoom.ini ^
  -savedir save ^
  -iwad iwads\doom2.wad ^
  -file mods\audio\IDKFAv2.wad ^
  -file mods\graphics\NeuralUpscale2x_v1.0.pk3
```

### Настройки GZDoom (gzdoom.ini)

Ключевые параметры:
- **Backend**: Vulkan (`vid_preferbackend=1`)
- **Разрешение**: 2752×1152, Fullscreen
- **Фильтрация текстур**: Anisotropic 16x
- **SSAO**: Включено (gl_ssao=1)
- **Широкоформатные графики**: Автозагрузка (`autoloadwidescreen=true`)
- **Язык**: Русский (`language=ru`)
- **Звук**: OpenAL, 48kHz, 128 каналов

## Возможности

- **Реальтайм Pitch Detection** — использует библиотеку `aubio` для сверхбыстрого определения частоты
- **Низкая задержка** — потоковая обработка аудио с блоками по 512 семплов (~11.6 мс)
- **Noise Gate** — порог шума для исключения ложных срабатываний
- **Pitch Smoothing** — медианный фильтр для стабилизации распознавания
- **Velocity Spike Detection** — автоматический выстрел при резкой атаке
- **Окно фокуса** — работает только когда GZDoom активен
- **Panic Button** — экстренная остановка клавишей Esc

## Использование

### 1. Настройка аудио

Подключите гитару к аудиоинтерфейсу и убедитесь, что Windows видит входное устройство. Уровень записи должен быть ~60-70% при обычной игре.

### 2. Выбор режима

Откройте `guitar_to_doom.py` и настройте режим:

```python
# Для электрогитары
INSTRUMENT_MODE = 'guitar'

# Для бас-гитары
INSTRUMENT_MODE = 'bass'
```

### 3. Запуск

```bash
python guitar_to_doom.py
```

### 4. Игра

Запустите GZDoom, перейдите в игровое окно и начинайте играть на гитаре!

## Управление

### Электрогитара (6 струн)

| Гитара | Клавиша | Действие |
|--------|---------|----------|
| E струна (6-я, ~82 Hz) | W | Идти вперёд |
| A струна (5-я, ~110 Hz) | S | Идти назад |
| D струна (4-я, ~147 Hz) | A | Стрейф влево |
| G струна (3-я, ~196 Hz) | D | Стрейф вправо |
| B струна (2-я, ~247 Hz) | ← | Поворот влево |
| e струна (1-я, ~330 Hz) | → | Поворот вправо |
| Слабый удар (со струной) | Space | Использовать / открыть дверь |
| Сильный удар (со струной) | Ctrl | Выстрел |
| **Palm mute** (без струны) | ] | Следующее оружие |
| Тишина | — | Остановка |
| **ESC** | — | **PANIC BUTTON** |

### Бас-гитара (4 струны)

| Бас | Клавиша | Действие |
|-----|---------|----------|
| E струна (4-я, ~41 Hz) | W | Идти вперёд |
| A струна (3-я, ~55 Hz) | S | Идти назад |
| D струна (2-я, ~73 Hz) | A | Стрейф влево |
| G струна (1-я, ~98 Hz) | D | Стрейф вправо |
| Слабый удар (со струной) | Space | Использовать / открыть дверь |
| Сильный удар (со струной) | Ctrl | Выстрел |
| **Palm mute** (без струны) | ] | Следующее оружие |
| **ESC** | — | **PANIC BUTTON** |

> **Palm mute** — приглуши струны ладонью и ударь. Volume spike есть, pitch нет → смена оружия.

## Калибровка

Каждый звукосниматель требует индивидуальной настройки. Откройте `guitar_to_doom.py` и отредактируйте секцию **КОНФИГУРАЦИЯ**:

```python
# Порог шума (0.001 - 0.1)
NOISE_GATE_THRESHOLD = 0.015

# Чувствительность выстрела
VELOCITY_SPIKE_THRESHOLD = 0.08

# Сглаживание pitch (2-8)
PITCH_SMOOTHING_WINDOW = 4
```

Подробное руководство по калибровке см. в [CALIBRATION.md](CALIBRATION.md).

## Troubleshooting

### Ложные срабатывания
Увеличьте `NOISE_GATE_THRESHOLD` в конфигурации.

### Большая задержка
Уменьшите `BLOCK_SIZE`, но ожидайте повышения нагрузки на CPU.

### Скрипт не реагирует на гитару
Проверьте настройки звука Windows и убедитесь, что входное устройство выбрано правильно.

### Не работает в GZDoom
Добавьте точное название окна в список `TARGET_WINDOW_TITLES`.
