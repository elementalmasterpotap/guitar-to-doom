# Калибровка Guitar-to-Doom Controller

## Установка зависимостей

```bash
# Стандартная установка
pip install -r requirements.txt

# Или по отдельности
pip install aubio sounddevice numpy pydirectinput pygetwindow pywin32 keyboard
```

## Подключение гитары

1. **Аудиоинтерфейс**: Подключите гитару через аудиоинтерфейс (рекомендуется) или адаптер Jack→USB
2. **Вход**: Используйте Hi-Z (Instrument) вход, если доступен
3. **Gain**: Настройте усиление так, чтобы при сильном ударе не было клиппинга

## Настройка в Windows

1. Откройте "Настройки звука" → "Ввод"
2. Убедитесь, что ваш аудиоинтерфейс выбран как устройство ввода
3. Уровень записи должен быть ~60-70% при обычной игре

## Калибровка параметров

### 1. Noise Gate Threshold (`NOISE_GATE_THRESHOLD`)

Настройка шумового порога:

```python
# Для активных звукоснимателей (humbucker)
NOISE_GATE_THRESHOLD = 0.02

# Для одиночных катушек (single-coil, более шумные)
NOISE_GATE_THRESHOLD = 0.015

# Для бас-гитары с высоким выходом
NOISE_GATE_THRESHOLD = 0.025
```

**Как настроить:**
1. Запустите скрипт
2. Не трогайте струны — смотрите на шкалу громкости
3. Если шкала показывает активность (бегут черные блоки) — увеличьте threshold
4. Если при лёгком прикосновении к струне нет реакции — уменьшите threshold

### 2. Velocity Spike Threshold (`VELOCITY_SPIKE_THRESHOLD`)

Настройка чувствительности выстрела:

```python
# Чувствительная (для лёгкой игры пальцами)
VELOCITY_SPIKE_THRESHOLD = 0.05

# Средняя (стандарт)
VELOCITY_SPIKE_THRESHOLD = 0.08

# Менее чувствительная (для агрессивной игры медиатором)
VELOCITY_SPIKE_THRESHOLD = 0.12
```

**Как настроить:**
1. Играйте обычную арпеджио
2. Если случайные выстрелы — увеличьте threshold
3. Если выстрел не срабатывает при резком ударе — уменьшите threshold

### 3. Pitch Smoothing (`PITCH_SMOOTHING_WINDOW`)

```python
# Быстрый отклик, возможны дребезг нот
PITCH_SMOOTHING_WINDOW = 2

# Стандартный баланс
PITCH_SMOOTHING_WINDOW = 4

# Очень плавный, менее отзывчивый
PITCH_SMOOTHING_WINDOW = 6
```

### 4. Частотные диапазоны струн

Если скрипт путает струны, отредактируйте диапазоны в `STRING_RANGES`:

```python
# Измерьте реальную частоту струн с помощью тюнера
# и отрегулируйте границы
'Струна': (частота - 10, частота + 10),
```

## Режимы Guitar/Bass

```python
# Для электрогитары (стандартный строй EADGBE)
INSTRUMENT_MODE = 'guitar'

# Для бас-гитары (стандартный строй EADG)
INSTRUMENT_MODE = 'bass'
```

## Запуск

```bash
# Перейдите в папку с GZDoom
cd C:\Games\gzdoom

# Запустите скрипт
python guitar_to_doom.py
```

## Типичные проблемы и решения

### Ложные срабатывания (движение без игры)
- Увеличьте `NOISE_GATE_THRESHOLD`
- Проверьте экранирование кабелей
- Используйте Noise Gate на аудиоинтерфейсе

### Задержка между ударом и действием
- Уменьшите `BLOCK_SIZE` (но CPU нагрузка возрастёт)
- Закройте другие программы
- Используйте ASIO драйвер (если доступен)

### Пропуск нот / нестабильное распознавание
- Увеличьте `PITCH_SMOOTHING_WINDOW`
- Проверьте настройку гитары
- Увеличьте громкость входного сигнала

### Скрипт не видит GZDoom
- Проверьте, что окно GZDoom активно
- Добавьте точное название окна в `TARGET_WINDOW_TITLES`

## Оптимизация задержки

| Параметр | Минимум | Стандарт | Максимум |
|----------|---------|----------|----------|
| BLOCK_SIZE | 256 (5.8ms) | 512 (11.6ms) | 1024 (23.2ms) |
| PITCH_SMOOTHING_WINDOW | 2 | 4 | 8 |

**Рекомендация**: Начните со стандартных значений и уменьшайте если ваш CPU справляется.

## Советы по игре

1. **Чистые ноты**: Играйте одну струну в момент времени для чёткого движения
2. **Выстрел**: Резкий удар по любой струне = выстрел
3. **Тишина**: Отпускание всех струн = остановка движения
4. **Panic Button**: Нажмите ESC если герой убегает куда не надо!
