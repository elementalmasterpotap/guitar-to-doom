# Патчноты — Guitar-to-Doom Controller Pro

## v2.1.0 — 2026-03-09

*Palm mute = смена оружия. Весь GZDoom теперь на гитаре без мыши.*

Долго не давал покоя один вопрос: как переключать оружие не отрываясь от струн?
Ответ оказался элегантным — palm mute. Приглушаешь струны ладонью, бьёшь — контроллер
видит volume spike без частоты и жмёт `]`. Тихо, натурально, не ломает ритм игры.

- Добавлен palm mute → `]` (следующее оружие) для обоих режимов
- Три уровня удара: слабый со струной → Space, сильный со струной → Ctrl, без струны → `]`
- Туториал обновлён: новый урок «Смена оружия» (гитара: урок 9, бас: урок 7)
- Схема управления в туториале показывает все три типа удара
- GUI лаунчер: добавлена мини-схема управления под карточками инструментов
- Архив с платными WAD'ами доступен на Telegraph

## v2.0.0 — 2026-03-09

*Рефакторинг: WinMM fallback + красивый GUI-лаунчер.*

- WinMM ctypes fallback — контроллер работает на **любой** версии Python, включая 3.15+
- GUI лаунчер (`GuitarDoomLauncher.exe`) — frameless окно, анимированный VU-метр,
  список всех игр, выбор инструмента, статусбар с аудио-бэкендом
- `build_exe.py` — автосборка EXE через PyInstaller (автоматически берёт Python 3.12)
- `setup.py` обновлён: сообщение что WinMM работает на любом Python
- `guitar_tutorial.py` — аналогичный WinMM fallback

## v1.0.0 — первый релиз

*Гитара управляет Doom через аудио-анализ.*

- YIN pitch detection (pure Python, без aubio)
- Noise gate, pitch smoothing, velocity spike detection
- Поддержка электрогитары (6 струн) и бас-гитары (4 струны)
- sounddevice backend + numpy fallback
- `guitar_tutorial.py` — интерактивное обучение
- Panic button (ESC)
